## Инструкция по запуску сайта
Шаги по запуску лендинга сайта локально на ПК:

1. **Скачайте и распакуйте** архив `project-landing.zip` в любую удобную директорию.
2. Перейдите в корень папки проекта.
3. Найдите файл `index.html`.
4. Далее ЛКМ и **«Открыть с помощью»** Chrome/другой браузер. 

## Структура проекта
В этом репозитории содержатся следующие компоненты:

*   **`components/`** — папка с компонентами интерфейса.
*   **`hooks/`** — правила поведения сайта (например, что делать при нажатии на кнопку «О проекте»)
*   **`lib/`** — сторонние библиотеки 
*   **`public/`** — статические ресурсы - это изображения, шрифты и иконки.
*   **`index.html`** — главная страница лендинга и точка входа в приложение.
*   **`styles.css`** — основные стили проекта.
*   **`package.json` / `pnpm-lock.yaml`** — конфигурационные файлы зависимостей.
